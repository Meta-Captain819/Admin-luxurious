To run the project locally:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/Meta-Captain819/Admin-luxurious.git
   cd Admin-luxurious
   ```

2. **Install dependencies:**

   ```bash
   npm install
   # or
   yarn install
   ```

   ```bash

   yarn dev
   ```

4. **Open your browser and navigate to:**

